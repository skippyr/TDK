CAPTION:
A reference of input characters of each keyboard key with modifier keys in a terminal.
They are used as to build the parse algorithm of the tdk_readkey() (3) function.

~RESERVED~: the key is reserved to perform an action from the terminal or operating system.
{...}     : a set of keys, usually correlated with a range.
...       : keys/characters were ommited, usually for convenience.
[BEG-END] : keys in range from BEG to END.
